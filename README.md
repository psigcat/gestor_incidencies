# Gestor d'incidncies
Gestor incidèncias genèric.
Inicialment es fa per gestionar les incidències de les capes de sanejament, però es crear per a gestionar altres infraestructures.

Aqust plugin funciona seleccionant les capes d'entrada.
Al clicar la icona del plugin revisa que hi hagi algun objecte seleccionat de les capes d'entrada.
Seguidament obre un formulari per crear una incidència i permet vincular diverses fotos a la mateixa incidència.
