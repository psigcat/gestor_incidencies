# Gestor d'incidències

Inicialment es fa per gestionar les incidències de les capes de sanejament, però es crear per a gestionar altres infraestructures.

Aquest plugin funciona seleccionant al menys un objecte de les capes d'entrada.

Al clicar la icona del plugin es obre un formulari per crear una incidència i permet vincular diverses fotos a la mateixa incidència.

Configuració de serveis i capes relacionades en el fitxer [conf.json](https://github.com/psigcat/gestor_incidencies/blob/main/config.json)
